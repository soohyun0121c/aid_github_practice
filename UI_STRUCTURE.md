## 🔄 화면 전환 흐름

```
홈 화면
  ├─ [SOS 테스트] → 다이얼로그 (메시지 미리보기)
  ├─ [지팡이 위치 보기] → 위치 탭으로 이동 안내
  └─ [자세히 보기] → SOS 기록 상세 (추후 구현)

위치 페이지
  ├─ [내 위치] → 지도 이동
  ├─ [지팡이 위치] → 지도 이동
  ├─ [길찾기] → 외부 지도 앱 실행
  ├─ [알림 보내기] → 지팡이에 알림 전송
  └─ [공유하기] → 위치 공유 (추후 구현)

SOS 페이지
  ├─ 긴급 연락처 탭
  │   ├─ [+ 추가] → 연락처 추가 다이얼로그
  │   ├─ [✏️ 편집] → 연락처 수정 다이얼로그
  │   └─ [🗑️ 삭제] → 삭제 확인 다이얼로그
  │
  └─ 주변 도움 요청 탭
      └─ [메시지 미리보기] → 미리보기 다이얼로그

설정 페이지
  ├─ [지팡이 연결하기] → 블루투스 연결 (추후 구현)
  ├─ [버튼 설정] → 동작 선택 다이얼로그
  ├─ [테스트 재생] → 음성 재생
  ├─ [진동 테스트] → 진동 발생
  ├─ [펌웨어 업데이트] → 업데이트 다이얼로그
  ├─ [설정 초기화] → 확인 다이얼로그
  └─ [앱 정보] → About 다이얼로그
```

## 🎨 색상 가이드

| 요소 | 색상 | 용도 |
|------|------|------|
| Primary | 파란색 | 일반 액션, 버튼 |
| Success | 초록색 | 연결됨, 정상 |
| Warning | 주황색 | 도움 요청, 주의 |
| Error | 빨간색 | 긴급, SOS, 연결 안 됨 |
| Info | 하늘색 | 정보, 안내 |
| Background | 회색 | 카드 배경 |

## 📐 디자인 원칙

1. **명확성**: 큰 버튼, 명확한 레이블
2. **일관성**: 모든 화면에서 동일한 패턴
3. **접근성**: 시각장애인을 위한 고대비 색상
4. **피드백**: 모든 액션에 대한 시각적 피드백 (SnackBar)
5. **계층**: Card를 활용한 정보 그룹화
